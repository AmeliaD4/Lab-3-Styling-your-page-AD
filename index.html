Css is Cascasing style Sheets, which allows you to style and choose how things look on your webpage. Internal CSS is code done within the tag itself, while extrernal Css is code done outside of the tag, in the actual css file. It also allows you to change the colors, size, font, and other things for several tags at once instead of just one tag itself.
